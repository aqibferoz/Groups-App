How to know if a verb belongs to Verbityyppi 5 ?


It groups all the verbs ending in the following way : -ita/-itä

Explanation

Following is a conjugation example with the verb häiritä (to disturb) :

Present tense: drop the a/ä and add se + add the termination . For 3rd pers. sing.: double the e
