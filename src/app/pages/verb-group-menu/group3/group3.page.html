Group 3
How to know if a verb belongs to Verbityyppi 3 ?


It groups all the verbs ending in the following way : -la/-lä, -na/-nä, -ra/-rä or -sta/-stä

Explanation

Following is a conjugation example with the verb nousta (to get up / to rise) :

Present tense: drop the 2 last letters and add an e + add the termination. For 3rd pers. sing.: double the e
